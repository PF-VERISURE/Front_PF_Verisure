## рҹҺӯ Roles y Funcionalidades

<details>
<summary><b>рҹ‘Ө Voluntario (EMPLOYEE)</b> вҖ” click para expandir</summary>

<br/>

El corazГіn de la plataforma. El voluntario puede:

- рҹ”Қ **Explorar** el catГЎlogo de proyectos publicados con filtros por categorГӯa, modalidad y ciudad
- вқӨпёҸ **Guardar favoritos** con un solo click
- рҹ“қ **Aplicar** a proyectos o unirse a lista de espera si estГЎn llenos
- рҹ“Ӣ **Gestionar** sus voluntariados activos y cancelar participaciГіn
- рҹҸҶ **Descargar certificados** digitales de cada proyecto completado
- рҹ”Қ Filtrar aplicaciones por estado: `PENDIENTE` В· `APROBADO` В· `LISTA DE ESPERA` В· `CERRADO`

</details>

<details>
<summary><b>рҹҸў ONG (ORGANIZATION)</b> вҖ” click para expandir</summary>

<br/>

La voz de las organizaciones. La ONG puede:

- вһ• **Crear proyectos** con formulario completo y validaciones en tiempo real
- рҹ“ё **Subir imagen** del proyecto (PNG/JPG/WEBP В· mГЎx. 10MB)
- рҹҢҚ **Configurar modalidad**: Presencial, Semipresencial o Virtual (con URL vГЎlida)
- рҹ“… **Gestionar fechas**, participantes requeridos y horas de dedicaciГіn
- рҹҺҜ **Categorizar** por ODS (Objetivos de Desarrollo Sostenible)
- рҹ“Ҡ **Ver estado** de cada proyecto: `EN REVISIГ“N` В· `PUBLICADO` В· `RECHAZADO` В· `CANCELADO`

</details>

<details>
<summary><b>рҹӣЎпёҸ Admin</b> вҖ” click para expandir</summary>

<br/>

El guardiГЎn del ecosistema. El admin puede:

- вң… **Aprobar o rechazar** proyectos enviados por las ONGs
- рҹ“Ҡ **Ver mГ©tricas** con grГЎficas de dona y barras por categorГӯa
- рҹ‘Ҙ **Gestionar perfiles** de ONGs y voluntarios
- рҹ“Ғ **Secciones especializadas**: En RevisiГіn В· Activos В· Archivados В· Rechazados
- рҹ‘ҒпёҸ **Ver participantes** por proyecto con modal detallado
